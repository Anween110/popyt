      Your YouTube Data API v3 token. Don't share this with anybody. It could be an API key or an OAuth 2.0 token.

    • Optional accessToken: string

      A Google OAuth 2.0 access token. Used for [[YouTube#oauth]] methods.

    • Default value options: YouTubeOptions = { cache: true, cacheTTL: 600, cacheCheckInterval: 600, cacheSearches: true }

      Caching options. Recommended to change.

    • Default value language: string = "en_US"

      The language for the API to respond in. See [[YouTube#getLanguages]].
